# Urban Environment Cartograms

## О проекте

В этом репозитории представлены картограммы, подготовленные в рамках исследовательской работы по анализу характеристик городской среды в муниципалитетах Московской области.

Проект посвящён пространственной визуализации экологически и социально значимых показателей городской среды для последующего аналитического сравнения территорий.

## Что представлено в репозитории (примеры картограмм)

- уровень автомобилизации населения;
- текущие эксплуатационные затраты на охрану окружающей среды;
- число спортивных сооружений;
- озеленённые территории общего пользования.

## Используемые инструменты

- Python
- Jupyter Notebook
- pandas
- геоаналитика и картографическая визуализация
